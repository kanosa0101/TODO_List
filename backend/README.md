# Todo Backend - Spring Boot

Java Spring Boot 后端服务，提供 RESTful API。

## 快速启动

### 使用 Maven

```bash
# 编译并运行
mvn spring-boot:run

# 或者先编译再运行
mvn clean package
java -jar target/todo-backend-1.0.0.jar
```

### 使用 IDE

1. 导入 Maven 项目
2. 运行 `TodoApplication.java` 主类

## 端口

默认运行在 `http://localhost:3001`
